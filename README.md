* ### demo_qsh中是简单的小样本的测试，数据只有new york times中的500条数据  
* ### origin_data是用完整的New York Times的数据  
* ### data是处理好的数据  
* ### data_process是处理数据的文件

* ### 实体识别中实体分为Begin，Inside，Outside，End，Single五类
